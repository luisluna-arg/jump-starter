JumpStarter

Windows logon command runner — quick reference

Configuration file

All commands are defined in appsettings.json. Click below to open it in your default editor.

Open appsettings.json

Config format

{
  "MaxConcurrentTasks": 5,
  "Commands": [
    {
      "Name": "Open Chrome",
      "Command": "chrome.exe https://www.google.com",
      "Delay": "00:00:00",
      "Shell": false
    },
    {
      "Name": "Maintenance script",
      "Command": "powershell.exe -NoProfile -File C:\\scripts\\startup.ps1",
      "Delay": "00:00:10",
      "Shell": true
    }
  ]
}
FieldTypeDescription
MaxConcurrentTasksnumberMaximum commands running simultaneously. Default: 5.
NamestringShort label shown in the tray tooltip.
CommandstringExecutable and arguments to run.
Delayhh:mm:ssWait time before this command starts. Default: "00:00:00".
Shellbooleanfalse — launch executable directly, no terminal window.
true — run via cmd.exe /c, hidden window (use for scripts, pipes). Default: false.
